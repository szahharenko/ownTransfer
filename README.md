# ownTransfer DEMO: 
https://agly.eu/

# CONFIGURATION:
Change CONFIG.PHP before use! No database required.

# CRON and auto remove:
To remove transfer automatically setup cron task for CRON.PHP, define period in configuration file - KEEP_DAYS parameter.
